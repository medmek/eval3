EvalMetrics 3

A propos

Cet application constitue le fruit de notre travail accompli dans le cadre du Projet de Fin d’Année du Département Génie Informatique au sein de l’Ecole Nationale des Sciences Appliquées d’Oujda.Lien ...L’objectif de ce projet est la mise en place d’une application web pour l’évaluation des petits projets informatiques. Cette application se base sur une classification proposée par le modèle ISO afin de rassembler les propriétés de la conception orientée objet combinées avec le modèle ISO. Cela pour pouvoir faire des tests sur les différentes phases de développement du projet.
